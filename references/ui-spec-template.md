# UI Spec 模板

## 1. 文档范围
- 本文档对应的产品或功能
- 本次覆盖的页面范围
- 不在本次 UI 范围内的页面或交互

## 2. 全局交互框架
- 终端类型：桌面端 / 移动端 / 响应式
- 默认入口
- 导航方式：顶部导航 / 侧边栏 / Tab / 单页流 / 其他
- 页面密度：轻量 / 标准 / 信息密集
- 全局交互原则

## 3. 页面清单
- 页面 1：页面目标
- 页面 2：页面目标
- 页面 3：页面目标

## 4. 页面级说明
对每个页面分别说明：

### 页面名称
- 页面目标
- 进入路径
- 适用角色
- 页面结构
- 主区域
- 辅助区域
- 默认展示内容
- 主操作
- 次操作
- 关键字段
- 与其他页面的跳转关系

## 5. 页面状态
- 零状态
- 空结果状态
- 加载态
- 错误态
- 成功反馈
- 权限受限状态

## 6. 关键交互
- 页面跳转关系
- 弹窗 / 抽屉 / 二级页面触发条件
- 表单提交流程
- 列表与详情联动关系
- 需要确认的高风险交互

## 7. 跨端与特殊约束
- 桌面端与移动端差异
- 权限差异
- 极端数据量下的展示约束
- 暂未定稿项

## 8. 原型提示词附录
只有用户明确要求给原型工具或设计 AI 的提示词时才写这一节。

- 页面范围
- 目标用户
- 导航结构
- 关键区域与组件
- 主次操作
- 必须覆盖的页面状态
- 不允许擅自扩展的内容
