# 线框说明模板

## 1. 原型范围
- 本次覆盖的页面
- 不在本次线框范围内的页面

## 2. 全局设计约束
- 终端类型：桌面端 / 移动端 / 响应式
- 导航方式：顶部导航 / 侧边栏 / Tab / 单页流
- 页面密度：轻量 / 标准 / 信息密集

## 3. 页面线框
对每个页面分别说明：

### 页面名称
- 页面目标
- 进入路径
- 页面结构
- 主区域
- 辅助区域
- 主操作
- 次操作
- 关键字段
- 状态变化

## 4. 状态设计
- 零状态
- 空结果状态
- 加载态
- 错误态
- 成功反馈

## 5. 关键交互
- 页面跳转关系
- 弹窗 / 抽屉 / 二级页面触发条件
- 表单提交流程
- 列表与详情联动关系

## 6. 特殊说明
- 权限差异
- 设备差异
- 暂未定稿项
